There are 500 instances for the SPP.
in101-200: M=500,  N=1000
in201-300: M=1000, N=1000
in401-500: M=1000, N=500
in501-600: M=1000, N=1500
in601-700: M=1500, N=1500

The input format is:
m n
Following n lines. The first number of each line i is the profit of bid i, 
following a number of integers, which are the jobs bid i covers.
